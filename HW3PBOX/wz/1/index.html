HW3PBOXWin7运行帮助

作者: Mxzsan | 日期: 2024年7月19日

首先,下载api-ms-win-core-path-l1-1-0.dll(如果提示丢失的话)

64位:

https://fel.forxhr.top:2022/down.php905b94c91755f2c2aec628c2131c7ffa.dll

32位:

https://fel.forxhr.top:2022/down.php/582edc1b799f2321cc476003b0fa5f98.dll

下载完成后,如果是64位系统,将api-ms-win-core-path-l1-1-0.dll拖入C:\Windows\System32文件夹,如果是32位系统,则拖入C:\Windows\SysWOW64文件夹

拖入完成后,可以看到已经没有丢失dll的提示了

然后,下载Visual C++ Redistributable for Visual Studio 2015

64位:

https://download.microsoft.com/download/9/3/F/93FCF1E7-E6A4-478B-96E7-D4B285925B00/vc_redist.x64.exe

32位:

https://download.microsoft.com/download/9/3/F/93FCF1E7-E6A4-478B-96E7-D4B285925B00/vc_redist.x86.exe

下载完成后,打开Visual C++ Redistributable for Visual Studio 2015安装程序

此时,你会看到这样一个页面:

勾选"我同意许可条款和条件(A)"

点击"安装(I)"

此时,有两种情况,一种是出现0x8007066错误,一种是安装成功

情况一 0x8007066错误

打开控制面板

点击"程序"

点击"程序和功能"

右键比Visual C++ Redistributable for Visual Studio 2015同版本或比Visual C++ Redistributable for Visual Studio 2015大的版本,点击"卸载(U)"

在弹出的窗口里,点击"卸载(U)"

出现"设置成功"即为卸载完成

之后重新打开安装程序,跟着之前的步骤安装即可

情况二 安装成功
